    //  $(e.currentTarget).css("backgroundColor","red");

    {
    //  $(this).css("backgroundColor","orange");

    }

    function mClick()
    {
        this.style.backgroundColor = "blue";
    }

Built in effects

  • fadeIn / fadeOut
  • slideUp / SlideDown

Callbacks

  • A function that is passed as an argument to another function.
  • Typically called after the second function has completed.

.animate()

  • Animates to a set of CSS properties
  • Not all properties can be animated
  • Performs animations asynchronously
  • Can take a callback function as last argument

Easing

  • A mathmatical description of an animation timeline
  • jQuery only comes with 2 types: linear and swing
  • Others can be added with plugins
  • You can write your own! (but you probably don't want to.)
